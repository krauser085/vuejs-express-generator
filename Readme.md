# Installation
```
$ npm install -g vuejs-express-generator
```

# Quick Start
Create your App:
```
$ vue-express { project directory }
```

Move to the directory of the App:
```
$ cd { project directory }
```

Install all dependancies:
```
$ npm install
```

Start Vue.js & Express application at http://localhost:3000 :
```
$ npm start
```
